Comandos Básicos do Linux

Comandos Úteis

  • ls: Lista arquivos e diretórios.
  • cd: Navega entre diretórios.
  • mkdir: Cria um novo diretório.
  • rm: Remove arquivos ou diretórios.
  • chmod: Modifica permissões de arquivos.

Navegando pelo Sistema de Arquivos
ls: Lista os arquivos e diretórios em um determinado local.
ls -l: Lista os arquivos com detalhes adicionais, como permissões, tamanho e data de modificação.
cd: Muda o diretório atual.
cd ..: Volta um diretório.
cd /: Vai para o diretório raiz.
pwd: Mostra o caminho completo do diretório atual.
Manipulando Arquivos e Diretórios
mkdir: Cria um novo diretório.
mkdir novo_diretorio: Cria um diretório chamado "novo_diretorio".
touch: Cria um arquivo vazio.
touch arquivo.txt: Cria um arquivo de texto chamado "arquivo.txt".
cp: Copia arquivos ou diretórios.
cp arquivo.txt novo_arquivo.txt: Copia o arquivo "arquivo.txt" para "novo_arquivo.txt".
mv: Move ou renomeia arquivos e diretórios.
mv arquivo.txt outro_diretorio: Move o arquivo para outro diretório.
rm: Remove arquivos ou diretórios.
ATENÇÃO: Use com cuidado! rm -r remove diretórios recursivamente.
rm arquivo.txt: Remove o arquivo "arquivo.txt".
Visualizando o Conteúdo de Arquivos
cat: Mostra o conteúdo de um arquivo na tela.
cat arquivo.txt: Mostra o conteúdo do arquivo "arquivo.txt".
less: Permite visualizar arquivos grandes página por página.
less arquivo.log: Abre o arquivo "arquivo.log" para visualização.
head: Mostra as primeiras linhas de um arquivo.
head -n 10 arquivo.log: Mostra as primeiras 10 linhas.
tail: Mostra as últimas linhas de um arquivo.
tail -n 5 arquivo.log: Mostra as últimas 5 linhas.
Gerenciando Permissões
chmod: Altera as permissões de arquivos e diretórios.
chmod 755 arquivo.sh: Define as permissões de execução para todos, leitura e execução para grupos e outros usuários.
Outros Comandos Úteis
sudo: Executa um comando com privilégios de superusuário (root).
sudo apt update: Atualiza a lista de pacotes disponíveis (no caso do gerenciador de pacotes apt).
man: Acessa os manuais de comando.
man ls: Mostra a página de manual do comando "ls".
find: Procura por arquivos e diretórios.
find / -name arquivo.txt: Procura por um arquivo chamado "arquivo.txt" em todo o sistema de arquivos.
grep: Procura por padrões em arquivos.
grep 'palavra' arquivo.txt: Procura pela palavra "palavra" no arquivo "arquivo.txt".
